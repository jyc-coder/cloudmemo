클라우드 메모장 API를 사용하여 저장 및 수정/삭제가 가능한 메모장을 구현함\

클라우드 메모장 API 깃허브 링크 : https://github.com/jyc-coder/cloudmemoapi

# 목표 기능
- 전체 메모 열람
- 단일 메모 열람
- 메모 생성/수정/삭제
- 웹 에디터 기능(`react-quill` 을 사용하여 구현함)
- 공유 기능 (링크 복사, 메일 전송)

# 기술 스택

- react, react-dom
- axios
- react-icons
- react-quill
- typescript
- styled-components
- styled-system
- react-copy-to-clipboard
- Navigator.share API
